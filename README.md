Impacto del COVID-19 en la Sociedad
Este proyecto es una página web informativa desarrollada con React y Vite, cuyo objetivo es mostrar el impacto del COVID-19 en la sociedad, incluyendo información introductoria, noticias relevantes de la época y datos sobre la mortandad causada por la pandemia.
La aplicación presenta una interfaz sencilla, organizada y adaptable, enfocada en la comprensión del contenido por parte del usuario.
Contenido del sitio
La página está dividida en varias secciones:
Inicio
Introducción general sobre el impacto social del COVID-19.
Noticias
Información relacionada con los primeros momentos de la pandemia en Ecuador, como la escasez de medicamentos y el aumento de casos.
Mortandad
Datos generales sobre fallecimientos y exceso de mortalidad durante la pandemia.
Tecnologías utilizadas
React – Biblioteca para la creación de interfaces de usuario
Vite – Herramienta de desarrollo rápido para proyectos web
JavaScript (ES6)
HTML5
CSS3
